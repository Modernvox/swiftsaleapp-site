SwiftSaleApp User Guide

Accurate steps for install, setup, live use, and manual fallback
← Back to Home Download PDF

Overview

SwiftSaleApp helps Whatnot sellers speed up packing and shipping with bin assignments, label annotation, and instant mailing lists. This guide reflects the current app behavior, including the manual fallback when live capture is not available.

1) Installation

  1. Download the Windows installer from the Downloads section.
  2. Run the installer (no admin needed; installs to your user AppData).
  3. Launch SwiftSaleApp from the Start Menu or Desktop shortcut.

If Windows SmartScreen warns you, choose “More info” → “Run anyway”.

2) Activation & Tiers

  1. On first launch, enter a valid email. Your install ID and tier will sync automatically when available. All users begin in free Starter tier on first launch.
  2. Paid users receive an activation code via email within 24 hours (typically ~14 hours).
  3. To enter a promo code, press Ctrl + Alt + D (dialog is labeled Enter Promo Code).
  • Starter: Free — up to 20 bins; includes core features (chat tools, sell rate, top buyers, label annotation, mailing list).
  • Bronze: $29.99/mo — up to 50 bins.
  • Silver: $69.99/mo — up to 150 bins.
  • Gold: $99.99/mo — up to 600 bins, 2-device limit per email, priority support.

Manage upgrades/downgrades on the website via Manage Subscription.

3) Capturing Buyers (Live + Manual Fallback)

Live capture (when enabled in your build): During a show, SwiftSaleApp can automatically detect buyer usernames and assign bins in real time.

Manual fallback (always available): If live capture is unavailable for any reason, simply copy & paste usernames into the Bidder List and SwiftSale will assign bins automatically. All downstream features (annotation, mailing list, reports) still work exactly the same.

Note: SwiftSaleApp does not require a Whatnot OAuth login. It works from live event data (when available) plus manual entry.

4) Bin Assignments

  1. As buyers win or purchase, each username gets a bin number (auto or manual).
  2. Assignments are idempotent — the same buyer keeps the same bin for the show.

5) Label Annotation (Post-Show)

  1. Download your Whatnot shipping labels PDF after the show.
  2. Open the PDF in SwiftSaleApp by clicking annotatle lables button. The app overlays buyer name and bin number on each label.
  3. Save or print the annotated PDF labelw. Matching labels to assigned bins in real time allows us to pack and ship much faster. Up to 57% faster to be exact!

6) Mailing List & Reports

  • Mailing List: SwiftSale auto creates a mailing list for you every time you annotate labels after each show. Existing customers are skipped and only their occuring spend is added to their profile. The next version of Swiftsale will include USPS integration so sellers can select one or all customers in their mailing list and auto create shipping labels to send promos & marketing materials. Customer info is your most valuable asset and ignoring the potential of an ongoing mailing list build is one of the biggest missed opportunities for future endeavors.
  • Top Buyers: Identify top customers per show and high spenders. Click on top buyers, A default message is copied. We can paste this message quickly into chat to engage our top buyers. Custom messages acan be set up in the settings tab. Set message and click save settings.
  • Sell-Rate: Quick snapshot of show performance. This tool shows you how many items seller is currently selling per minute and also gives an accurate estimation of how many items will be sold at the current rate over the length of a 2 or 3 hour show.

7) Device Limits & Sync

  • Gold: Enforced 2-device limit per email. If you replace a device, contact support to release an old one.
  • Install/tier syncing runs automatically based on your email and subscription status. All users begin in Free starter mode on first run.

8) Shortcuts & Best Practices

  • Ctrl + Alt + D: Open Enter Promo Code dialog.
  • Keep buyer usernames consistent. Swiftsale App V4 includes a warning feature that changes the Add Bidder button from green to yellow when the same Whatnot username is pasted consecutively. This allows the seller or operator to quickly double check they have copied the current username correctly. When the Add Bidder turns yellow the app still records the transaction, but allows the operator to change if needed. Since the username was already assigned a bin during the show and it may be just a duplicated paste error no harm is done to your bin assigments. You simply paste the correct winning username and Swiftsale will assign the next available bin number and the Add Bidder button will automatically reset to green. Again, it's just a warning to alert you incase you didn't copy the winning buyers username as expected.

9) Support

Questions or need a device reset? Email support@swiftsaleapp.com. Include your Install ID from the app’s status bar for faster help. Gold users receive priority response.

Activation codes are emailed within 24 hours of payment (typically ~14 hours).
