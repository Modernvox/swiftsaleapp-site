SwiftSaleApp User Guide

Accurate steps for install, setup, live use, and manual fallback
← Back to Home Download PDF

Overview

SwiftSaleApp helps Whatnot sellers speed up packing and shipping with bin assignments, label annotation, and instant mailing lists. This guide reflects the current app behavior, including the manual fallback when live capture is not available.

1) Installation

  1. Download the Windows installer from the Downloads section.
  2. Run the installer (no admin needed; installs to your user AppData).
  3. Launch SwiftSaleApp from the Start Menu or Desktop shortcut.

If Windows SmartScreen warns you, choose “More info” → “Run anyway”.

2) Activation & Tiers

  1. On first launch, enter a valid email. Your install ID and tier will sync automatically when available.
  2. Paid users receive an activation code via email within 24 hours (typically ~14 hours).
  3. To enter a promo or unlock code, press Ctrl + Alt + D (dialog is labeled Enter Promo Code).
  • Starter: Free — up to 20 bins; includes core features (chat tools, sell rate, top buyers, label annotation, mailing list).
  • Bronze: $29.99/mo — up to 50 bins.
  • Silver: $69.99/mo — up to 150 bins.
  • Gold: $99.99/mo — up to 600 bins, 2-device limit per email, priority support.

Manage upgrades/downgrades on the website via Manage Subscription.

3) Capturing Buyers (Live + Manual Fallback)

Live capture (when enabled in your build): During a show, SwiftSaleApp can automatically detect buyer usernames and assign bins in real time.

Manual fallback (always available): If live capture is unavailable for any reason, simply type buyer usernames into the Bidder List and assign bins manually. All downstream features (annotation, mailing list, reports) still work exactly the same.

Note: SwiftSaleApp does not require a Whatnot OAuth login. It works from live event data (when available) plus manual entry.

4) Bin Assignments

  1. As buyers win or purchase, each username gets a bin number (auto or manual).
  2. Assignments are idempotent — the same buyer keeps the same bin for the show.
  3. You can view, edit, or lock bins if needed before you print labels.

5) Label Annotation (Post-Show)

  1. Download your Whatnot shipping labels PDF after the show.
  2. Open the PDF in SwiftSaleApp. The app overlays buyer name and bin number on each label.
  3. Save or print the annotated PDF. Match labels to bins and pack faster.

Tip: If a buyer was added manually, they will still annotate correctly so long as their username matches the order.

6) Mailing List & Reports

  • Mailing List: Export a CSV of buyers for marketing and follow-ups.
  • Top Buyers: Identify repeat customers and high spenders.
  • Sell-Rate: Quick snapshot of show performance.

7) Device Limits & Sync

  • Gold: Enforced 2-device limit per email. If you replace a device, contact support to release an old one.
  • Install/tier syncing runs automatically based on your email and activation status.

8) Shortcuts & Best Practices

  • Ctrl + Alt + D: Open Enter Promo Code dialog.
  • Keep buyer usernames consistent. If manual entry is used, match the exact Whatnot username to ensure perfect annotation.
  • Before printing, spot-check a few labels to confirm annotation (bin and buyer overlay) is correct.

9) Support

Questions or need a device reset? Email support@swiftsaleapp.com. Include your Install ID from the app’s status bar for faster help. Gold users receive priority response.

Activation codes are emailed within 24 hours of payment (typically ~14 hours).
