FAQ

Questions sellers ask before they switch.

Clear answers for sellers who need reliability, speed, and something their team can actually use live.

Does SwiftSale work during the live show?

Yes. SwiftSale is built for real-time use while the show is running, including winner capture, bin assignment, and game-related workflows.

Can I use it after the show for fulfillment?

Yes. That is one of the biggest advantages: your data is already organized, so label annotation, mailing lists, and exports are much faster.

Do my buyers see the game features?

They see the result on your stream or screen output. SwiftSale helps you create a more engaging show experience without slowing your workflow behind the scenes.

Does each install start fresh?

Yes. Each user has their own local app data by default, so a new install starts clean unless you intentionally migrate or sync data.

What operating system does it support?

The current production release is for Windows desktop.

How do updates work?

SwiftSale supports in-app auto-updates, so when new versions are published, your users can move forward without hunting for manual files.
