FAQ

Questions first-time visitors usually ask.

Clear answers in plain English.

What is a BIN?

A BIN is the spot where a buyer's order goes while you are organizing and packing.

Does SwiftSale work during the show?

Yes. SwiftSale helps you capture winners, assign bins, and run engagement features while the show is happening.

What does label annotation mean?

SwiftSale adds buyer and BIN details to packing slips so orders are easier to identify while packing.

Does SwiftSale print shipping labels?

No. SwiftSale helps annotate packing slips so fulfillment is easier and faster to follow.

Are games post-sale only?

Yes. Bingo, Bin Bonanza, and Prize Vault are post-sale bonus engagement features and do not affect the purchase itself.

What does it run on?

The current production version is for Windows desktop.
